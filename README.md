## 🛠️ Tecnologías y herramientas aprendidas

Durante mis proyectos y experiencias académicas, he trabajado con diversas tecnologías y herramientas, enfocadas en **análisis de datos, desarrollo y aprendizaje automático**:  

- **Lenguajes de programación:** Python, R, Java, SQL/MySQL, PHP
- **Data Science y Machine Learning:** análisis estadístico, predicción académica, modelado de datos, regresión, clasificación y clustering  
- **Deep Learning (básico) y NPL:** redes neuronales simples, procesamiento de lenguaje natural y análisis de texto  
- **Bases de datos y backend:** procedimientos, triggers, CRUD y gestión de información  
- **Librerías y frameworks principales:**  
  - `Pandas`, `NumPy`, `Matplotlib` → análisis y visualización de datos  
  - `scikit-learn` → Machine Learning  
  - `TensorFlow` / `Keras` → Deep Learning básico  
  - `NetworkX` → modelado de grafos y optimización  
  - `Jupyter Notebook` → documentación interactiva y ejecución de código
